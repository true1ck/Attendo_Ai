Setup & Configuration

Getting Started

1
Configure OneDrive Integration

Set up your OneDrive connection for cloud file storage and synchronization.

# In your .env file ONEDRIVE_CLIENT_ID=your_client_id ONEDRIVE_CLIENT_SECRET=your_client_secret ONEDRIVE_REDIRECT_URI=your_redirect_uri
2
Set Excel Export Paths

Configure the local and cloud paths for Excel file exports.

# Excel sync configuration EXCEL_EXPORT_PATH=./exports/excel/ ONEDRIVE_FOLDER_PATH=/ATTENDO/Reports/
3
Initialize Excel Templates

Set up your Excel templates with proper formatting and Power Automate triggers.

Required Excel Table Structure:

  • employee_id - Vendor/Employee identifier
  • date - Attendance date
  • status - Work status (WFH/WFO/Leave/etc.)
  • location - Work location
  • timestamp - Record creation time
  • manager_approval - Approval status
4
Test the Connection

Verify that the Excel sync is working correctly with a test export.

Run Test Export

Troubleshooting Guide

Common Issues

üö´ Sync Not Working
Possible Causes:
  • OneDrive authentication expired
  • Network connectivity issues
  • File permissions problems
  • Excel file is locked/open
Solutions:
  • Re-authenticate OneDrive connection
  • Check network connectivity
  • Verify file permissions
  • Close Excel files before sync
‚ö†Ô∏è Power Automate Not Triggering
Possible Causes:
  • Excel table structure changed
  • Power Automate flow disabled
  • Trigger conditions not met
  • Flow quota exceeded
Solutions:
  • Verify Excel table schema
  • Enable Power Automate flow
  • Review trigger conditions
  • Check Power Automate quotas
üìä Data Missing or Incorrect
Possible Causes:
  • Sync timing issues
  • Data filtering problems
  • Template formatting errors
  • Partial sync completion
Solutions:
  • Force a complete resync
  • Review data filters
  • Validate Excel templates
  • Check sync completion logs

Getting Help
