Getting Started as Administrator

Welcome to ATTENDO Administrator Portal! As an admin, you have complete control over the system configuration, user management, and operational oversight.

Your Responsibilities

  • Manage all users (Vendors, Managers, Admins)
  • Configure system settings and policies
  • Set up and maintain holiday calendars
  • Perform bulk data imports and exports
  • Handle reconciliation and billing corrections
  • Monitor system health and audit logs
  • Generate executive reports
  • Manage notification configurations
  • Ensure data security and backups

Default Admin Credentials

  • Username: Admin
  • Default Password: admin123
  • Email: admin@attendo.com
Security Alert: Change the default admin password immediately after first login. Enable two-factor authentication for admin accounts.

User Management

Comprehensive user administration for all system roles.

User Roles & Permissions

Role Key Permissions Access Level
Admin Full system control, all configurations Unrestricted
Manager Team management, approvals, reports Team-level
Vendor Submit status, view own records Self-only

Creating New Users

  1. Navigate to User Management section
  2. Click "Add New User" button
  3. Select user role (Vendor/Manager/Admin)
  4. Enter required information:
    • Username (unique identifier)
    • Email address
    • Full name
    • Department/Company
    • Initial password
  5. Assign to manager (for vendors)
  6. Set activation status
  7. Click "Create User"

User Account Actions

  • Activate/Deactivate: Enable or disable user access
  • Reset Password: Force password reset on next login
  • Change Role: Modify user permissions
  • Update Profile: Edit user information
  • Delete User: Permanently remove (with data archival)
Admin Tip: Always deactivate users instead of deleting to maintain audit trail integrity.

Administrator FAQs

Q: How do I reset a user's password?

A: Go to User Management, find the user, click "Actions" ‚Üí "Reset Password". The user will be prompted to set a new password on next login.

Q: Can I recover deleted data?

A: Deleted data is archived for 90 days by default. Use the Data Recovery tool or restore from backup if needed.

Q: How do I handle mass leave during emergencies?

A: Use Bulk Operations ‚Üí Emergency Leave to mark all affected vendors. This creates proper audit trails and notifications.

Q: What's the maximum file size for bulk import?

A: The system supports files up to 10MB. For larger imports, split into multiple files or use the API.

Q: How do I configure email notifications?

A: Go to System Settings ‚Üí Notification Configuration ‚Üí Email Settings. Enter SMTP details and test the connection.

Q: Can I customize the dashboard for different departments?

A: Yes, use Dashboard Configuration to create department-specific views with relevant widgets and metrics.

Q: How often should I perform system maintenance?

A: Monthly maintenance is recommended: clear old logs, optimize database, update security patches, and verify backups.

Q: What happens when a manager leaves?

A: Reassign their team to another manager before deactivating. Use Teams & Groups ‚Üí Bulk Reassign for efficiency.
