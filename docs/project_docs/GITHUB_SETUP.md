```bash

cd "G:\Projects\Hackathon\MediaTek\vendor-timesheet-tool"

# Add the remote origin (replace YOUR_USERNAME with your actual GitHub username)

# Set the main branch

# Push your code to GitHub
git push -u origin main
```
